Bracketology – Report – Dashboard

Bracketology aims to examine the balance between NCAA men's basketball regular-season offensive/defensive metrics and how that correlates with postseason [March Madness] success.

The technical portion of this project features pre-processing using python script, database creation using Postgres/pgAdmin 4, interactive query scripting to design one large query for data extraction, and dashboard design in Tableau Public.
